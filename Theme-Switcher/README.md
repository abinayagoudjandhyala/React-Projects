# Theme Switcher

This is a simple React project where I created a theme switcher that allows users to toggle between light mode and dark mode.

By building this project, I learned how to use React state and how to update the UI dynamically based on user actions. I also understood how conditional styling works in React.

In this project, clicking a button changes the theme of the page and updates the background and text colors accordingly.

This project helped me practice React basics such as functional components, state management using useState, and handling user interactions.
