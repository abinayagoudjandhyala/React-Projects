# Todo App

This is a simple Todo App built using React. The app allows users to add tasks, mark them as completed, and delete them.

While doing this project, I learned how to use the useState hook to manage multiple states like input value and list of tasks. I also understood how to update arrays in React using map and filter.

In this project, I implemented adding new tasks, deleting tasks, and marking tasks as completed by clicking on them. I also practiced conditional styling to show completed tasks with a line-through effect.

This project helped me understand how to handle lists, events, and dynamic UI updates in React.
