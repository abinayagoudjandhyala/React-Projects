# React Projects

This repository contains the React projects I built while learning React. I created these projects to practice and improve my understanding of frontend development.

In this repository, I worked on multiple small projects like Profile Card, Theme Switcher, Feedback Form, Todo App, and Blog App.

By doing these projects, I learned how React works and how to build user interfaces using components. I understood how to use useState for managing data and how to handle user input and events.

I also learned how to work with lists using map, remove items using filter, and update UI dynamically. In the Blog App, I learned routing and how to navigate between different pages using react-router.

Overall, these projects helped me understand component structure, state management, event handling, and how to build simple real-world applications using React.
